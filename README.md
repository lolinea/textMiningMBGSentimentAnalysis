There are multiple data such as csv, json, and txt. Below are the explanation for each file: 

1. combined_slang_words.txt: a file obtained from kaggle, it contains the list of slangs often used by people in Indonesia. The purpose is to replace the slangs into a formal word
2. data.csv: The combined file from 3 (DatasetMBG.csv) and 4 (tweets.csv)
3. DatasetMBG.csv: dataset obtained from kaggle, contains scraped data from X regarding people's posts on X
4. tweets.csv: dataset obtained from scraping on X
5. slang_indo.csv: similar to 1 (combined_slang_words.txt), contains list of slangs.
6. slangs.json: combination of 1 (combined_slang_words.txt) and 5 (slang_indo.csv)
7. tweets_labeled_indoroberta.csv: dataset that has been given sentiment label
8. tweetsLabeledFixed.csv: Final dataset, contains tweet_id, username, tweet_content, and tweet_label that has been adjusted and checked manually.
